### 1. তলৰ কোনকেইটা ৰাশি এটা চলকযুক্ত বহুপদ আৰু কোনকেইটা নহয়? তোমাৰ উত্তৰৰ যুক্তি দিয়া। {#ex-2.1:qno1}

{{<flex marginY="1rem" alignItems="center">}}
<span>@(i) 4x^2 - 3x + 7@</span>
<span>@(ii) y^2 + \sqrt{2}@</span>
<span>@(iii) 3\sqrt{t} + t\sqrt{2}@</span>
<span>@(iv) y + \dfrac{2}{y}@</span>
<span>@(v) x^{10} + y^3 + z^{50}@</span>
{{</flex>}}

@\underline{\text{সমাধান:}}@

**@(i) 4x^2 - 3x + 7@**

**উত্তৰ:** এই ৰাশিটো এটা চলকযুক্ত বহুপদ ৰাশি। কাৰণ, ইয়াত চলকৰ সংখ্যা @1@ আৰু চলক @x@ ৰ সূচক এটা পূৰ্ণ সংখ্যা।

**@(ii) y^2 + \sqrt{2}@**

**উত্তৰ:** এই ৰাশিটো এটা চলকযুক্ত বহুপদ ৰাশি। কাৰণ, ইয়াত চলকৰ সংখ্যা @1@ আৰু চলক @y@ ৰ সূচক এটা পূৰ্ণ সংখ্যা।

**@(iii) 3\sqrt{t} + t\sqrt{2}@**

**উত্তৰ:** এই ৰাশিটো এটা চলকযুক্ত বহুপদ ৰাশি নহয়। কাৰণ, ইয়াত চলকৰ সংখ্যা @1@ যদিও ইয়াত চলক @t@ ৰ সূচক @\frac{1}{2}@ টো পূৰ্ণ সংখ্যা নহয়।

**@(iv) y + \frac{2}{y}@**

**উত্তৰ:** এই ৰাশিটো এটা চলকযুক্ত বহুপদ ৰাশি নহয়। কাৰণ ইয়াত চলকৰ সংখ্যা @1@ যদিও ইয়াত চলক @y@ ৰ সূচক @-2@ টো পূৰ্ণ সংখ্যা নহয়।

**@(v) x^{10} + y^3 + t^{50}@**

**উত্তৰ:** এই ৰাশিটো এটা চলকযুক্ত বহুপদ ৰাশি। কাৰণ, এই বহুপদ ৰাশিটোত তিনিটা ভিন্ন চলক আছে।

## অনুশীলনী 2.2 {#ex-2.2}

### 1. @5x - 4x^2 + 3@ বহুপদৰ মান নিৰ্ণয় কৰা যেতিয়া - {#ex-2.2:qno1}

{{<flex marginY="1rem" alignItems="center">}}
<span>@(i) x = 0@</span>
<span>@(ii) x = -1@</span>
<span>@(iii) x = 2@</span>
{{</flex>}}

@\underline{\text{উত্তৰ:}}@

**(i) @x = 0@**

**সমাধান:**

@
\begin{align}
p(x) &= 5x - 4x^2 + 3 \\
p(0) &= 5(0) - 4(0)^2 + 3 \\
&= 0 - 0 + 3 \\
&= 3
\end{align}
@

**(ii) @x = -1@**

**সমাধান:**

@
\begin{align}
p(x) &= 5x - 4x^2 + 3 \\
p(-1) &= 5(-1) - 4(-1)^2 + 3 \\
&= -5 - 4(1) + 3 \\
&= -5 - 4 + 3 \\
&= -6
\end{align}
@

**(iii) @x = 2@**

**সমাধান:**

@
\begin{align}
p(x) &= 5x - 4x^2 + 3 \\
p(2) &= 5(2) - 4(2)^2 + 3 \\
&= 10 - 4(4) + 3 \\
&= 10 - 16 + 3 \\
&= -3
\end{align}
@

### 2. তলৰ বহুপদবোৰৰ প্ৰত্যেকৰ বাবে @p(0), p(1)@ আৰু @p(2)@ নিৰ্ণয় কৰা। {#ex-2.2:qno2}

{{<flex marginY="1rem" alignItems="center">}}
<span>@(i) p(y) = y^2 - y + 1@</span>
<span>@(ii) p(t) = 2 + t + 2t^2 - t^3@</span>
<span>@(iii) p(x) = x^3@</span>
<span>@(iv) p(x) = (x - 1)(x + 1)@</span>
{{</flex>}}

@\underline{\text{উত্তৰ:}}@

**(i) @p(y) = y^2 - y + 1@**

**সমাধান:**

@
\begin{align}
p(0) &= 0^2 - 0 + 1 = 1 \\
p(1) &= 1^2 - 1 + 1 = 1 \\
p(2) &= 2^2 - 2 + 1 = 3
\end{align}
@

**(ii) @p(t) = 2 + t + 2t^2 - t^3@**

**সমাধান:**

@
\begin{align}
p(0) &= 2 + 0 + 2(0)^2 - 0^3 = 2 \\
p(1) &= 2 + 1 + 2(1)^2 - 1^3 = 4 \\
p(2) &= 2 + 2 + 2(2)^2 - 2^3 = 4
\end{align}
@

**(iii) @p(x) = x^3@**

**সমাধান:**

@
\begin{align}
p(0) &= 0^3 = 0 \\
p(1) &= 1^3 = 1 \\
p(2) &= 2^3 = 8
\end{align}
@

### 3. কাষত উল্লেখিত মানবোৰ বহুপদটোৰ শূণ্য হয়নে নহয় সত্যাপন কৰি চোৱা। {#ex-2.2:qno3}

{{<flex marginY="2rem" alignItems="center">}}
<span>@(i) p(x) = 3x + 1, x = -\frac{1}{3}@</span>
<span>@(ii) p(x) = 5x - \pi, x = \frac{4}{\pi}@</span>
<span>@(iii) p(x) = x^2 - 1, x = 1, -1@</span>
<span>@(iv) p(x) = (x + 1)(x - 2), x = -1, 2@</span>
<span>@(v) p(x) = x^2, x = 0@</span>
<span>@(vi) p(x) = lx + m, x = -\frac{m}{l}@</span>
<span>@(vii) p(x) = 3x^2 - 1, x = -\frac{1}{\sqrt{3}}, \frac{2}{\sqrt{3}}@</span>
<span>@(viii) p(x) = 2x + 1, x = \frac{1}{2}@</span>
{{</flex>}}

@\underline{\text{উত্তৰ:}}@

{{<scrollable>}}

**(i) @p(x) = 3x + 1, x = -\frac{1}{3}@**

@
\begin{align}
p(x) & = 3x + 1 \\
p(-\frac{1}{3}) & = 3(-\frac{1}{3}) + 1 \\
& = -1 + 1 \\
& = 0
\end{align}
@

{{</scrollable>}}

{{<scrollable>}}

**(ii) @p(x) = 5x - \pi, x = \frac{4}{\pi}@**

@
\begin{align}
p(x) &= 5x - \pi \\
p(\frac{4}{\pi}) &= 5(\frac{4}{\pi}) - \pi \\
&= \frac{20}{\pi} - \pi \\
&= \frac{20 - \pi^2}{\pi}
\end{align}
@

{{</scrollable>}}

{{<scrollable>}}

**(iii) @p(x) = x^2 - 1, x = 1, -1@**

@
\begin{align}
p(x) &= x^2 - 1 \\
p(1) &= 1^2 - 1 = 0 \\
p(-1) &= (-1)^2 - 1 = 0
\end{align}
@

{{</scrollable>}}

{{<scrollable>}}

**(iv) @p(x) = (x + 1)(x - 2), x = -1, 2@**

@
\begin{align}
p(x) &= (x + 1)(x - 2) \\
p(-1) &= (-1 + 1)(-1 - 2) = 0 \\
p(2) &= (2 + 1)(2 - 2) = 0
\end{align}
@

{{</scrollable>}}

{{<scrollable>}}

**(v) @p(x) = x^2, x = 0@**

@
\begin{align}
p(x) &= x^2 \\
p(0) &= 0^2 = 0
\end{align}
@

{{</scrollable>}}

{{<scrollable>}}

**(vi) @p(x) = lx + m, x = -\frac{m}{l}@**

@
\begin{align}
p(x) &= lx + m \\
p(-\frac{m}{l}) &= l(-\frac{m}{l}) + m \\
&= -m + m \\
&= 0
\end{align}
@

{{</scrollable>}}

{{<scrollable>}}

**(vii) @p(x) = 3x^2 - 1, x = -\frac{1}{\sqrt{3}}, \frac{2}{\sqrt{3}}@**

@
\begin{align}
p(x) &= 3x^2 - 1 \\
p(-\frac{1}{\sqrt{3}}) &= 3(-\frac{1}{\sqrt{3}})^2 - 1 = 3(\frac{1}{3}) - 1 = 0 \\
p(\frac{2}{\sqrt{3}}) &= 3(\frac{2}{\sqrt{3}})^2 - 1 = 3(\frac{4}{3}) - 1 = 3
\end{align}
@

{{</scrollable>}}

{{<scrollable>}}

**(viii) @p(x) = 2x + 1, x = \frac{1}{2}@**

@
\begin{align}
p(x) &= 2x + 1 \\
p(\frac{1}{2}) &= 2(\frac{1}{2}) + 1 = 1 + 1 = 2
\end{align}
@

{{</scrollable>}}

### 4. তলৰ প্ৰতিটো বহুপদৰ শূণ্য নিৰ্ণয় কৰা। {#ex-2.2:qno4}

{{<flex marginY="1rem" alignItems="center">}}
<span>@(i) p(x) = x + 5@</span>
<span>@(ii) p(x) = x - 5@</span>
<span>@(iii) p(x) = 2x + 5@</span>
<span>@(iv) p(x) = 3x - 2@</span>
<span>@(v) p(x) = 3x@</span>
<span>@(vi) p(x) = ax, a \ne 0@</span>
<span>@(vii) p(x) = cx + d, c \ne 0, c, d \text{ বাস্তৱ সংখ্যা}@</span>
{{</flex>}}

@\underline{\text{উত্তৰ:}}@

**(i) @p(x) = x + 5@**

@
\begin{align}
p(x) &= x + 5 \\
0 &= x + 5 \\
x &= -5
\end{align}
@

@\therefore p(x) = x + 5@ ৰ শূণ্য হ'ল @x = -5@।

**(ii) @p(x) = x - 5@**

@
\begin{align}
p(x) &= x - 5 \\
0 &= x - 5 \\
x &= 5
\end{align}
@

@\therefore p(x) = x - 5@ ৰ শূণ্য হ'ল @x = 5@।

**(iii) @p(x) = 2x + 5@**

@
\begin{align}
p(x) &= 2x + 5 \\
0 &= 2x + 5 \\
2x &= -5 \\
x &= -\frac{5}{2}
\end{align}
@

@\therefore p(x) = 2x + 5@ ৰ শূণ্য হ'ল @x = -\frac{5}{2}@।

**(iv) @p(x) = 3x - 2@**

@
\begin{align}
p(x) &= 3x - 2 \\
0 &= 3x - 2 \\
3x &= 2 \\
x &= \frac{2}{3}
\end{align}
@

@\therefore p(x) = 3x - 2@ ৰ শূণ্য হ'ল @x = \frac{2}{3}@।

**(v) @p(x) = 3x@**

@
\begin{align}
p(x) &= 3x \\
0 &= 3x \\
x &= 0
\end{align}
@

@\therefore p(x) = 3x@ ৰ শূণ্য হ'ল @x = 0@।

**(vi) @p(x) = ax@**

@
\begin{align}
p(x) &= ax \\
0 &= ax \\
x &= 0
\end{align}
@

@\therefore p(x) = ax@ ৰ শূণ্য হ'ল @x = 0@।

**(vii) @p(x) = cx + d@**

@
\begin{align}
p(x) &= cx + d \\
0 &= cx + d \\
cx &= -d \\
x &= -\frac{d}{c}
\end{align}
@

@\therefore p(x) = cx + d@ ৰ শূণ্য হ'ল @x = -\frac{d}{c}@।

## অনুশীলনী 2.3 {#ex-2.3}

### 1. @x^3 + 3x^2 + 3x + 1@ ক তলৰ বহুপদেৰে হৰণ কৰিলে পোৱা ভাগশেষ নিৰ্ণয় কৰা : {#ex-2.3:qno1}

{{<flex marginY="1rem" alignItems="center">}}
<span>@(i) x + 1@</span>
<span>@(ii) x - \frac{1}{2}@</span>
<span>@(iii) x@</span>
<span>@(iv) x + \pi@</span>
<span>@(v) 5 + 2x@</span>
{{</flex>}}

@\underline{\text{উত্তৰ:}}@

**(i) @x + 1@**

**সমাধান:**

{{<scrollable>}}

@
\require{enclose}
\begin{array}{rll}
x^2 + 2x + 1 && \\[-3pt]
x + 1 \enclose{longdiv}{x^3 + 3x^2 + 3x + 1}\kern-.2ex \\[-3pt]
\underline{-(x^3 + x^2)} \phantom{+ 3x + 1} && \\[-3pt]
2x^2 + 3x \phantom{+ 1} && \\[-3pt]
\underline{-(2x^2 + 2x)} \phantom{+ 1} && \\[-3pt]
x + 1 && \\[-3pt]
\underline{-(x + 1)} && \\[-3pt]
0 &&
\end{array}
@

ভাগশেষ @0@

{{</scrollable>}}

**(ii) @x - \frac{1}{2}@**

**সমাধান:**

{{<scrollable>}}

@
\begin{array}{rll}
x^2 + \frac{7}{2}x + \frac{19}{4} && \\[-3pt]
x - \frac{1}{2} \enclose{longdiv}{x^3 + 3x^2 + 3x + 1}\kern-.2ex \\[-3pt]
\underline{-(x^3 - \frac{1}{2}x^2)} \phantom{+ 3x + 1} && \\[-3pt]
\frac{7}{2}x^2 + 3x \phantom{+ 1} && \\[-3pt]
\underline{-(\frac{7}{2}x^2 - \frac{7}{4}x)} \phantom{+ 1} && \\[-3pt]
\frac{19}{4}x + 1 && \\[-3pt]
\underline{-(\frac{19}{4}x - \frac{19}{8})} && \\[-3pt]
\frac{27}{8} &&
\end{array}
@

ভাগশেষ @\frac{27}{8}@

{{</scrollable>}}

**(iii) @x@**

**সমাধান:**

{{<scrollable>}}

@
\begin{array}{rll}
x^2 + 3x + 3 && \\[-3pt]
x \enclose{longdiv}{x^3 + 3x^2 + 3x + 1}\kern-.2ex \\[-3pt]
\underline{-x^3} \phantom{+ 3x^2 + 3x + 1} && \\[-3pt]
3x^2 + 3x \phantom{+ 1} && \\[-3pt]
\underline{-3x^2} \phantom{+ 3x + 1} && \\[-3pt]
3x + 1 && \\[-3pt]
\underline{-3x} \phantom{+ 1} && \\[-3pt]
1 &&
\end{array}
@

ভাগশেষ @1@

{{</scrollable>}}

**(iv) @x + \pi@**

**সমাধান:**

{{<scrollable>}}

@
\begin{array}{l}
\phantom{x+\pi)\;x^3+{}} x^2 + (3-\pi)x + (3-3\pi+\pi^2) \\[-3pt]
x + \pi \enclose{longdiv}{x^3 + 3x^2 + 3x + 1}\kern-.2ex \\[-3pt]
\phantom{x+\pi)\;} \underline{-(x^3 + \pi x^2)} \\[-3pt]

### 5. উৎপাদক বিশ্লেষণ কৰা : {#ex-2.4:qno5}

{{<flex direction="col" marginY="2rem">}}
<span>@(i) x^3 - 2x^2 - x + 2@</span>
<span>@(ii) x^3 - 3x^2 - 9x - 5@</span>
<span>@(iii) x^3 + 13x^2 + 32x + 20@</span>
<span>@(iv) 2y^3 + y^2 - 2y - 1@</span>
<span>@(v) x^3 + x^2 - x - 1@</span>
<span>@(vi) x^3 + x^2 + x + 1@</span>
<span>@(vii) x^3 + 2x^2 - x - 12@</span>
<span>@(viii) x^3 + 3x^2 - 7x - 6@</span>
<span>@(ix) 3x^3 + 5x^2 - 16x - 2@</span>
{{</flex>}}

@\underline{\text{সমাধান:}}@

**(i) @x^3 - 2x^2 - x + 2@**

{{<scrollable>}}
ধৰাহ'ল @p(x) = x^3 - 2x^2 - x + 2@  
@2@ ৰ উৎপাদকবোৰ হ'ল @\pm 1, \pm 2@।  
@x = 1@ লৈ পাওঁ,  
@p(1) = 1^3 - 2(1)^2 - 1 + 2 = 1 - 2 - 1 + 2 = 0@  
@\therefore (x - 1)@ ক @p(x)@ ৰ এটা উৎপাদক।  

এতিয়া,  
@
\begin{align}
&= x^3 - 2x^2 - x + 2 \\ 
&= x^2(x - 1) - x(x - 1) - 2(x - 1) \\
&= (x - 1)(x^2 - x - 2) \\
&= (x - 1)(x^2 - 2x + x - 2) \\
&= (x - 1)\{x(x - 2) + 1(x - 2)\} \\
&= (x - 1)(x - 2)(x + 1)
\end{align}
@

{{</scrollable>}}

**(ii) @x^3 - 3x^2 - 9x - 5@**

{{<scrollable>}}
ধৰাহ'ল @p(x) = x^3 - 3x^2 - 9x - 5@  
@-5@ ৰ উৎপাদকবোৰ হ'ল @\pm 1, \pm 5@।  
@x = -1@ লৈ পাওঁ,  
@p(-1) = (-1)^3 - 3(-1)^2 - 9(-1) - 5 = -1 - 3 + 9 - 5 = 0@  
@\therefore (x + 1)@ ক @p(x)@ ৰ এটা উৎপাদক।  

এতিয়া,  
@
\begin{align}
&= x^3 - 3x^2 - 9x - 5 \\
&= x^2(x + 1) - 4x(x + 1) - 5(x + 1) \\
&= (x + 1)(x^2 - 4x - 5) \\
&= (x + 1)(x^2 - 5x + x - 5) \\
&= (x + 1)\{x(x - 5) + 1(x - 5)\} \\
&= (x + 1)(x - 5)(x + 1) \\
&= (x + 1)^2(x - 5)
\end{align}
@

{{</scrollable>}}

**(iii) @x^3 + 13x^2 + 32x + 20@**

{{<scrollable>}}
ধৰাহ'ল @p(x) = x^3 + 13x^2 + 32x + 20@  
@20@ ৰ উৎপাদকবোৰ হ'ল @\pm 1, \pm 2, \pm 4, \pm 5, \dots@  
@x = -1@ লৈ পাওঁ,  
@p(-1) = (-1)^3 + 13(-1)^2 + 32(-1) + 20 = -1 + 13 - 32 + 20 = 0@  
@\therefore (x + 1)@ ক @p(x)@ ৰ এটা উৎপাদক।  

এতিয়া,  
@
\begin{align}
&= x^3 + 13x^2 + 32x + 20 \\
&= x^2(x + 1) + 12x(x + 1) + 20(x + 1) \\
&= (x + 1)(x^2 + 12x + 20) \\
&= (x + 1)(x^2 + 10x + 2x + 20) \\
&= (x + 1)\{x(x + 10) + 2(x + 10)\} \\
&= (x + 1)(x + 10)(x + 2)
\end{align}
@

{{</scrollable>}}

**(iv) @2y^3 + y^2 - 2y - 1@**

{{<scrollable>}}
ধৰাহ'ল @p(y) = 2y^3 + y^2 - 2y - 1@  
@y = 1@ লৈ পাওঁ,  
@p(1) = 2(1)^3 + (1)^2 - 2(1) - 1 = 2 + 1 - 2 - 1 = 0@  
@\therefore (y - 1)@ ক @p(y)@ ৰ এটা উৎপাদক।  

এতিয়া,  
@
\begin{align}
&= 2y^3 + y^2 - 2y - 1 \\
&= 2y^2(y - 1) + 3y(y - 1) + 1(y - 1) \\
&= (y - 1)(2y^2 + 3y + 1) \\
&= (y - 1)(2y^2 + 2y + y + 1) \\
&= (y - 1)\{2y(y + 1) + 1(y + 1)\} \\
&= (y - 1)(y + 1)(2y + 1)
\end{align}
@

{{</scrollable>}}

**(v) @x^3 + x^2 - x - 1@**

{{<scrollable>}}

ধৰাহ'ল, @p(x) = x^3 + x^2 - x - 1@  
@x = 1@ লৈ পাওঁ,  
@p(1) = 1^3 + 1^2 - 1 - 1 = 1 + 1 - 1 - 1 = 0@  
@\therefore (x - 1)@ ক @p(x)@ ৰ এটা উৎপাদক।  

@
\begin{align}
&= x^3 + x^2 - x - 1 \\
&= x^3 - x^2 + 2x^2 - 2x + x - 1 \\
&= x^2(x - 1) + 2x(x - 1) + 1(x - 1) \\
&= (x - 1)(x^2 + 2x + 1) \\
&= (x - 1)(x + 1)^2
\end{align}
@

{{</scrollable>}}

**(vi) @x^3 + x^2 + x + 1@**

{{<scrollable>}}

ধৰাহ'ল, @p(x) = x^3 + x^2 + x + 1@  
@x = -1@ লৈ পাওঁ,  
@p(-1) = (-1)^3 + (-1)^2 + (-1) + 1 = -1 + 1 - 1 + 1 = 0@  
@\therefore (x + 1)@ ক @p(x)@ ৰ এটা উৎপাদক।  

@
\begin{align}
&= x^3 + x^2 + x + 1 \\
&= x^2(x + 1) + 1(x + 1) \\
&= (x + 1)(x^2 + 1)
\end{align}
@

{{</scrollable>}}

**(viii) @x^3 + 3x^2 - 7x - 6@**

{{<scrollable>}}

ধৰাহ'ল @p(x) = x^3 + 3x^2 - 7x - 6@  
@x = 2@ লৈ পাওঁ,  
@p(2) = 2^3 + 3(2)^2 - 7(2) - 6 = 8 + 12 - 14 - 6 = 0@  
@\therefore (x - 2)@ ক @p(x)@ ৰ এটা উৎপাদক।  

এতিয়া,  
@
\begin{align}
&= x^3 + 3x^2 - 7x - 6 \\
&= x^2(x - 2) + 5x(x - 2) + 3(x - 2) \\
&= (x - 2)(x^2 + 5x + 3)
\end{align}
@

{{</scrollable>}}

*(টোকা: (vii) আৰু (ix) নম্বৰ প্রশ্নবোৰত উৎপাদক বিশ্লেষণ কৰিব পৰা নগ'ল, সম্ভৱত প্রশ্নবোৰত মুদ্রণ জনিত ভুল থাকিব পাৰে।)*

### 6. যদি @x^2 + px + q@ আৰু @x^2 + mx + n@ বহুপদ দুটাৰ @x + a@ এটা সাধাৰণ উৎপাদক, তেন্তে প্ৰমাণ কৰা যে- @a = \frac{n - q}{p - m}@ {#ex-2.4:qno6}

{{<scrollable>}}

@\underline{\text{সমাধান:}}@

ধৰাহ'ল @f(x) = x^2 + px + q@ আৰু @g(x) = x^2 + mx + n@  
@\because x + a@ দুয়োটাৰে এটা সাধাৰণ উৎপাদক, গতিকে @f(-a) = 0@ আৰু @g(-a) = 0@।

@
\begin{align}
\therefore (-a)^2 + p(-a) + q &= 0 \\
\Rightarrow a^2 - ap + q &= 0 \quad \dots(1)
\end{align}
@

আৰু

@
\begin{align}
(-a)^2 + m(-a) + n &= 0 \\
\Rightarrow a^2 - am + n &= 0 \quad \dots(2)
\end{align}
@

(1) আৰু (2) ৰ পৰা পাওঁ,

@
\begin{align}
a^2 - ap + q &= a^2 - am + n \\
\Rightarrow -ap + am &= n - q \\
\Rightarrow a(m - p) &= n - q \\
\Rightarrow a &= \frac{n - q}{m - p} \\
\Rightarrow a &= \frac{-(q - n)}{-(p - m)} \\
\Rightarrow a &= \frac{q - n}{p - m}
\end{align}
@

{{</scrollable>}}

বিঃদ্ৰঃ: প্রশ্নত দিয়া উত্তৰটো পাবলৈ হ'লে উৎপাদকটো @x - a@ হ'ব লাগিব। কিন্তু প্রশ্নত @x + a@ দিয়া আছে বাবে উত্তৰত @n - q@ ৰ ঠাইত @q - n@ পোৱা গৈছে।

### 6. তলৰ ঘনকেইটা বিস্তাৰ কৰি লিখা : {#ex-2.5:qno6}

{{<flex marginY="1rem" alignItems="center">}}

<span>@(ii) (2a - 3b)^3@</span>
<span>@(iii) (\frac{3}{2}x + 1)^3@</span>
<span>@(iv) (x - \frac{2}{3}y)^3@</span>
{{</flex>}}

@\underline{\text{সমাধান:}}@

{{<scrollable>}}

**(i) @(2x + 1)^3@**

@
\begin{align}
&= (2x + 1)^3 \\
&= (2x)^3 + 1^3 + 3(2x)(1)(2x + 1) \\
&= 8x^3 + 1 + 6x(2x + 1) \\

### 13. (i) যদি @x + y + z = 0@, তেন্তে দেখুওৱা যে @x^3 + y^3 + z^3 = 3xyz@ {#ex-2.5:qno13-i}

#### (ii) যদি @a + b + c = 0@, তেন্তে দেখুওৱা যে @a^2(b+c) + b^2(c+a) + c^2(a+b) + 3abc = 0@ {#ex-2.5:qno13-ii}

#### (iii) যদি @2a - b + c = 0@, তেন্তে দেখুওৱা যে @4a^2 - b^2 + c^2 + 4ac = 0@ {#ex-2.5:qno13-iii}

#### (iv) যদি @a + b + c = 0@, তেন্তে দেখুওৱা যে @\frac{(b+c)^2}{bc} + \frac{(c+a)^2}{ca} + \frac{(a+b)^2}{ab} = 3@ {#ex-2.5:qno13-iv}

#### (v) যদি @a^2 + b^2 + c^2 - ab - bc - ca = 0@, তেন্তে দেখুওৱা যে @a = b = c@ {#ex-2.5:qno13-v}

{{</scrollable>}}

{{<scrollable>}}

**(i) যদি @x + y + z = 0@, তেন্তে দেখুওৱা যে @x^3 + y^3 + z^3 = 3xyz@**

@\underline{\text{সমাধান:}}@

আমি জানো যে,

@x^3 + y^3 + z^3 - 3xyz = (x + y + z)(x^2 + y^2 + z^2 - xy - yz - zx)@

এতিয়া, যদি @x + y + z = 0@ হয়, তেন্তে

@
\begin{align}
&= x^3 + y^3 + z^3 - 3xyz \\
&= 0 \times (x^2 + y^2 + z^2 - xy - yz - zx) \\
&\Rightarrow x^3 + y^3 + z^3 - 3xyz = 0 \\
& \Rightarrow x^3 + y^3 + z^3 = 3xyz
\end{align}
@

(দেখুওৱা হ'ল)

{{</scrollable>}}

{{<scrollable>}}

**(ii) যদি @a + b + c = 0@, তেন্তে দেখুওৱা যে @a^2(b+c) + b^2(c+a) + c^2(a+b) + 3abc = 0@**

@\underline{\text{সমাধান:}}@

দিয়া আছে, @a + b + c = 0@

@\Rightarrow b + c = -a, \; c + a = -b, \; a + b = -c@

@
\begin{align}
\text{বাওঁপক্ষ} &= a^2(b+c) + b^2(c+a) + c^2(a+b) + 3abc \\
&= a^2(-a) + b^2(-b) + c^2(-c) + 3abc \\
&= -a^3 - b^3 - c^3 + 3abc \\
&= -(a^3 + b^3 + c^3 - 3abc) \\
&= -(0) \quad [\because a + b + c = 0] \\
&= 0 \\
&= \text{সোঁপক্ষ}
\end{align}
@

(দেখুওৱা হ'ল)

{{</scrollable>}}

{{<scrollable>}}

**(iii) যদি @2a - b + c = 0@, তেন্তে দেখুওৱা যে @4a^2 - b^2 + c^2 + 4ac = 0@**

@\underline{\text{সমাধান:}}@

দিয়া আছে,

@
\begin{align}
& 2a - b + c = 0 \\
\Rightarrow \; & b = 2a + c \\
\Rightarrow \; & b^2 = (2a + c)^2 \\
\Rightarrow \; & b^2 = 4a^2 + 4ac + c^2 \\
\Rightarrow \; & 4a^2 - b^2 + c^2 + 4ac = 0
\end{align}
@

(দেখুওৱা হ'ল)

{{</scrollable>}}

{{<scrollable>}}

**(iv) যদি @a + b + c = 0@, তেন্তে দেখুওৱা যে @\frac{(b+c)^2}{bc} + \frac{(c+a)^2}{ca} + \frac{(a+b)^2}{ab} = 3@**

@\underline{\text{সমাধান:}}@

দিয়া আছে,

@a + b + c = 0@  
@\therefore@ @b + c = -a@, @c + a = -b@, @a + b = -c@

@
\begin{align}
\text{বাওঁপক্ষ} &= \frac{(b+c)^2}{bc} + \frac{(c+a)^2}{ca} + \frac{(a+b)^2}{ab} \\
&= \frac{(-a)^2}{bc} + \frac{(-b)^2}{ca} + \frac{(-c)^2}{ab} \\
&= \frac{a^2}{bc} + \frac{b^2}{ca} + \frac{c^2}{ab} \\
&= \frac{a^3 + b^3 + c^3}{abc} \\
&= \frac{3abc}{abc} \quad [\because a + b + c = 0] \\
&= 3 \\
&= \text{সোঁপক্ষ}
\end{align}
@

(দেখুওৱা হ'ল)

{{</scrollable>}}

{{<scrollable>}}

**(v) যদি @a^2 + b^2 + c^2 - ab - bc - ca = 0@, তেন্তে দেখুওৱা যে @a = b = c@**

@\underline{\text{সমাধান:}}@

দিয়া আছে, 
@a^2 + b^2 + c^2 - ab - bc - ca = 0@

দুয়োপক্ষক 2 ৰে পূৰণ কৰি পাওঁ,

@
\begin{align}
& 2(a^2 + b^2 + c^2 - ab - bc - ca) = 2 \times 0 \\
\Rightarrow \; & 2a^2 + 2b^2 + 2c^2 - 2ab - 2bc - 2ca = 0 \\
\Rightarrow \; & (a^2 - 2ab + b^2) + (b^2 - 2bc + c^2) + (c^2 - 2ca + a^2) = 0 \\
\Rightarrow \; & (a - b)^2 + (b - c)^2 + (c - a)^2 = 0
\end{align}
@

যিহেতু বৰ্গ ৰাশিৰ সমষ্টি শূণ্য, গতিকে প্ৰতিটো ৰাশিৰ মান শূণ্য হ'ব।

@
\begin{align}
\therefore & \; (a - b)^2 = 0 \\
& \Rightarrow a - b = 0 \\
& \Rightarrow a = b
\end{align}
@

ঠিক একেদৰে,

@b = c@ আৰু @c = a@

@\therefore a = b = c@

(দেখুওৱা হ'ল)
